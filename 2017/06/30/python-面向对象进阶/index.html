四个实现自省的函数

2.1.1 hasattr

hasattr(object,name)

判断object是否有名字是name的属性。
等价于 name in object.__dict__


2.1.2 getattr

getattr(object, name, default=None)

获取object名称空间里的”name”属性的值，如果存在，就返回该值；不存在，就返回default。
等价于：object.name (object.__dict__["name"])


2.1.3 setattr

setattr(object, name, value)

修改对象object的name属性值为value。
等价于：object.name=value


2.1.4 delattr

delattr(object,name)

删除object对象的名称空间下的名叫name的属性
等价于：del object.name

7 __slots__

     __slots__是一个类变量，变量值可以是列表、元组或其他可迭代的对象，也可以是一个字符串(意味着只有一个数据属性)
     通常，我们写在__init__()里的属性，都会存在对象的__dict__里。大量数据时，会占用比较多的内存。__slots__就是为了解决这个问题。
     只有书写了__slots__，该类的对象就不再创建自己的名称空间。同时也不能在添加__slots__元组以外的变量名，可以做到限制对象属性的作用。

结果：

     下面，添加了__init__()也没有对象名称空间。此时的__init__()只能用来初始化。

总结一下__slots__的作用：

  1. 节省内存
  2. 限制对象的属性
    用处：用在那些需要多个对象，且对象的属性都是固定的。

11 __del__

     析构方法，当对象在内存中被释放时，自动触发执行。
注：此方法一般无须定义，因为Python是一门高级语言，程序员在使用时无需关心内存的分配和释放，因为此工作都是交给Python解释器来执行，所以，析构函数的调用是由解释器在进行垃圾回收时自动触发执行的。（python3手动调用，不起作用）
     析构函数里写的内容，是在删除对象以前需要执行的。

结果：

     程序结束时，f1与内存中的Foo的绑定也会解除，只有内存中的Foo()引用计数为0，才会执行__del__，并且定义了__del__的实例无法被Python的循环垃圾收集器收集，所以尽量不要自定义__del__。一般情况下，__del__不会破坏垃圾处理器。

     如果类没有定义析构函数，那么当引用计数为0时，并不会自动清除，而是要等到gc的下一个周期回收。

一个类应该由哪几个部分组成？

     类要有标识它的类名，类要有继承关系，类还要有自己的变量属性和函数属性，而这两种属性在python中都被存储在类的名称空间里。
     所以，python中一个类有3个部分组成：类名，继承关系，类的名称空间（字典）。

为什么Mytype里的__init__会自动运行？
